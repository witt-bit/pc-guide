## 软件包

### 1.`open`

### (1).下载安装

> 适合所有发行版Linux、Mac

```shell
wget https://raw.githubusercontent.com/witt-bit/pc-guide/refs/heads/master/linux/scripts/open.sh

chmod +x open.sh
./open.sh
```

### (2).AUR安装

> 适合Arch Linux、Manjaro
> 地址: https://aur.archlinux.org/packages/open

```shell
yay -S open
```
